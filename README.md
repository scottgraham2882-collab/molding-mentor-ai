# AI Molding Coach

Injection molding training, troubleshooting, and scientific molding education platform.

## Mission

Molding Mentor AI helps operators, technicians, supervisors, and managers learn injection molding, troubleshoot problems using structured thinking, preserve manufacturing knowledge, and collaborate effectively to improve manufacturing processes.

## Core Principles

- **Help people learn:** Teach the “why,” not just the “what.”
- **Help people troubleshoot:** Use structured thinking and evidence-based problem solving.
- **Preserve knowledge:** Capture lessons before they are lost.
- **Keep it simple:** Make tools fast, practical, and easy to use on the shop floor.
- **Support collaboration:** Good troubleshooting combines experience, data, and teamwork.
- **Practice the 5 Whys:** Ask why, gather evidence, rule out possibilities, think critically, and seek another perspective when you reach the limit of your knowledge.
- **Choose humility before ego:** Think for yourself, learn all you can, and ask for help before pride becomes the problem.
- **Put people first:** Technology exists to help people grow, learn, succeed, and become better problem-solvers, teachers, leaders, and teammates.

## Getting Started

Install dependencies and start the Next.js development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the mobile-first troubleshooting coach homepage.
